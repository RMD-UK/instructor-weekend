Group Practice: The Learning Conversation

The aim of this session is to allow candidates to practice providing feedback using the techniques of the learning conversation. There are 45 minutes allocated to this session.

Pre-session requirements
  • Arrange teaching space (chairs and space)
  • Set up and check manikin
  • Set up and check AED
Running the session
Stage / RoleInstructions
Manikin set up BY INSTRUCTOR BEFORE SESSIONLungs inflate without problem Clicker turned off Blue pad underneath manikin to kneel on
AED set up BY INSTRUCTOR BEFORE SESSIONTurns on (batteries OK) Pads present and sticky Correct scenario selected
IntroductionInform candidates of session aim Candidates will take turns to take on role of ‘student’ and ‘instructor’. The ‘student’ will demonstrate an element of the BLS algorithm, after which the ‘instructor’ will feedback on the students’ performance using the learning conversation. RMD faculty will then provide feedback on how the learning conversation was used.
Briefing and scenarioSelect first instructor and student pair Assign an aspect of the algorithm to be demonstrated from: Demonstrate safe approach and call for help Check for breathing and briefing help Demonstration of chest compressions and rescue breaths Demonstration of AED use and delivery of a shock Instructor and student pair demonstrate the above. Following the Student completing the scenario, the instructor should then feedback on their performance using the learning conversation Note: The student should aim to demonstrate the skill to their best ability. In a particularly strong instructor candidate group, where there are no concerns about BLS provision, the session lead can consider introducing ‘intentional errors’ in the students’ performance for the purposes of feedback – but this would be an exception rather than a rule for this session, the preference being to allow ICs to perfect BLS techniques throughout day 1 of the course.
Brief candidates remaining in groupThey should watch candidate performance and decide on elements they would feedback on to the candidate
FeedbackFollowing the end of the scenario, and the conclusion of the ICs feedback; Group session leads should then feedback on the scenario, using the learning conversation, including the whole IC group in the feedback. Ensure at this point that all errors made by the ‘student’ (intentional and unintentional) are fed back to the whole group at this point so unintentional mistakes aren’t accidentally adopted by other ICs.
GroupRepeat the scenario to allow all ICs present the opportunity to practice working with the learning conversation
