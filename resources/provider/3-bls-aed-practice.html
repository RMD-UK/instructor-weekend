Bls/Aed Practice

The aim of this session is to allow candidates to practice performing BLS with guidance & assess if they have the required standard to continue on the course. The standard expected is that necessary to pass the BLS Provider Course.

Pre-session requirements
  • Attend lecture and demonstration of BLS/AED
  • Manikins & automated external defibrillators in teaching rooms
  • Clean & working
  • Cleaning wipes within rooms
Running the session
Stage / RoleInstructions
Manikin set upClicker turned on to start with Blue pad underneath manikin to kneel on
Prepare & OpenIntroduce instructors Introduce candidates Explain the importance of this session as they will be teaching this to others so their skills need to be of a very high quality Explain what will happen in the session
Facilitate and, as necessary, interacts with the co-instructor to confirm assessment decisions.Manikin in clear space where the candidates can see what is going on Instructor explains and demonstrates the skill Facilitate candidate practice while acknowledging prior experience Assess whether learning outcomes are met Additional coaching as needed
Facilitate/assess itDivide group within teaching rooms Each candidate then performs BLS Instructor gives scenario. You must include some scenarios that are unsafe (varying degrees of danger). Some should be so dangerous that the candidate must not approach the patient Make sure set is put back each time but NOT whilst giving feedback. Use the learning conversation for feedback. Use multiple scenarios (see programme) If student has done multiple things to improve on, tell them about the major 3 and leave the rest for their next turn. DO NOT give them to the student all at once
Conclusion/CloseBring the group back together Any questions about the session Explain where to go to for break Closure: Make sure these are the last things you say to the candidates 3 points about the session drawing on the most important points for the group e.g learnt the correct sequence of BLS, importance of correct chest compressions and important of clear movements so their students can see what to do Tell candidates if they have been successful, or not, before the leave the teaching room
EndTidy room, clean manikins Instructor debrief Provide feedback about every candidate via the course feedback form
